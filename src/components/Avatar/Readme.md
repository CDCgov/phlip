#### Avatar example with initials
```jsx
<Avatar
  userName="Spongebob Squarepants"
  initials="SS"
/>
```

#### Card avatar example with initials and custom style
```jsx
<Avatar
  cardAvatar
  big
  initials="BS"
  userName="Buffy Summers"
  style={{
    backgroundColor: 'lightBlue',
    color: 'black',
    borderColor: 'black'
  }}
/>
```
